Experience

Developer Intern
Novacoast Inc.

At Novacoast, I've been working on a specific project called Full Resolution Health - a HIPAA compliant, cloud-based app that helps autism clinics share videos and feedback with their patients. I worked on designing and onboarding a new scoring feature. I initially worked on mockups for the iOS app and the scoring feature on the web app, which I then coded based on the iterated mockups. I'm currently working on the iOS app. I've also worked on some backend features for the admin tools and implemented Google Analytics.

Web Assistant Intern
DIVE

DIVE was an early startup that delivered groceries in Isla Vista on demand via their iPhone app. I worked on setting up their e-commerce on their website so consumers could order directly from the website.

Intern
OUST Labs Inc.

OUST Labs was an educational, Quiz Up like app that I joined in the early stages of its founding. I worked on the initial requirements and wireframes for the iOS app. My goals at OUST were to improve the user interface and user experience.

Curriculum Director
PathtoExcel

I worked to create curriculum that adhered to Common Core Standards for the Project Based Learning (PBL) nonprofit, PathtoExcel. I also worked to help set up and improve the user experience of the website.
