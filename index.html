Side Projects

  • Cloud Classroom


    Cloud classroom was made to become a "virtual classroom". It simulates a classroom environment by making it easy for students to communicate with the teacher and goes a step further to allow nondistracting interactions between students. Teachers can host a class with a virtual whiteboard that students can see in real time (hosted on the AWW app). Teachers can also connect a twitch stream and host a chat room all in one page where students can join and interact with each other and teachers alike. The chat stream was written natively with node.js servers and differentiates student and teachers. It also has features that allow students to effectively queue questions directed at the teacher which the teacher can then address.


    Javascript/HTML/Firebase

  • Camera Hacks


    A camera that focuses on accessibility for the blind (irony intended). This was based off of the Microsoft question, how would you design X (usually Microsoft Excel, PPT etc.) for the blind, but after further inspection I realized there was no Android accessibility for the native camera app. The app uses Google ML Kit to recognize objects, and Google text to voice to read aloud what is seen, and speech to text to recognize commands like, “Take a picture” or "What am I looking at?".


    Javascript/HTML/Firebase

  • Cue It!


    An android app that allows user to create/join rooms in realtime via Firebase realtime database. If a user decides to create a room, a 4 digit code is generated so that other users could join that room with no sign up necessary. Once a user joins a room, they can add watch and add Youtube videos to a queue. Other users in the room can upvote videos and the queue will be updated in realtime.


    Javascript/HTML/Firebase

  • Workday Career Fair


    An additional feature to Workday’s android app that used optical character recognition through Google MLKit to allow recruiters to scan resumes and parse viable information in real time. The feature also allowed recruiters to add notes, a scoring system, and team assignments to possible contenders. Recruiters could also create separate folders for different career fairs/schools/occupations Placed in top 5 at Workday's Intern Hackathon


    Android/Java
